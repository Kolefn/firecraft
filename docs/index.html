collection

This class is a high level representation of a firestore collection. Path parameters wrapped in curly brackets are supported.

new collection(path: string)
Parameters
path (string) is a path wich may include parameters.
Example
new collection("users/{userId}/characters/{characterId}/achievements");

extractPathParams

Parses a path string and returns a 'true' value map of the parameters encapsulated by {}.

extractPathParams(string: string): object
Parameters
string (string = "") is the path string
Returns
object: a 'true' value map of parameter names.

injectPathArgs

Creates a new path with the encapsulated parameters replaced with arguments.

injectPathArgs(path: string, args: object, nameMap: object): string
Parameters
path (string = "") is the path with potential parameters
args (object = {}) is a map of values to replace parameters with.
nameMap (object = {}) is an optional reference map for when arg keys do not match relevant param name.
Returns
string: a new path with data in place of parameters.
Example
util.injectPathArgs('col/{docId}/list/{itemId}', {userId: 'zach123', achievementId: 'highscore'}, {docId: 'userId', itemId: 'achievementId'});

bool: true if the document path is valid.
