collection

This class is a high level representation of a firestore collection. Specifically, it represents a generic reference to a collection before arguments are provided for the document ID parameters in the path.

new collection(path: string)
Parameters
path (string) is a generic path wich may include document ID parameters.
Example
new collection("users/{userId}/characters/{characterId}/achievements");
